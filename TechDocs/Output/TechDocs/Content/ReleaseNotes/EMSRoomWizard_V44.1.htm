What's New for Version 44.1

Enhancement

Area

Description

Checking In

Checking in From Any EMS Product Will Register on the EMS Room Wizard Panel

A check-in on any EMS product is registered on the EMS Room Wizard Panel. Once a user checks into a booking using any EMS product, the "Start" button is removed from the Room Wizard panel so that user does not need to check in again from the Room Wizard panel.

Note: The "Start Early" button will still appear before the meeting start time even if the user has checked in from any EMS Product. Room Wizard currently does not provide an API to suppress this button.

Registration and Rooms

DATABASE AUTHENTICATION USING ACTIVE DIRECTORY FAILS WHEN EMSUSER WAS DISABLED

When Active Directory (AD) was used for database authentication and EMSUser was disabled on the SQL server, an error occurred.

Fix: Modified functionality so that users can disable EMSUser and continue to authenticate successfully in this manner.

Start Early

Start Early Option Now Checks Users In To Meetings

The Start Early option in EMS Room Wizard now registers a check-in and appears per the configuration of the device used (minutes prior to start time). No Shows configured for the location do not register if Start Early is checked.

Note: This only works on the latest Room Wizard firmware version as Start Early did not exist on older versions.
