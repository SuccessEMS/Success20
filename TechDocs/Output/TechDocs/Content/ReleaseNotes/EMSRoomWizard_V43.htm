Add a Room

Adding a Room May Cause "You have reached your maximum room count of 0" Error

Some customers experienced this error and had to re-enter their EMS Room Wizard license.

Fix: Modified functionality so that the error no longer occurs and customers can add new rooms as expected.

06.02.2016

Client

43.0.384

Fix

Area

Description

Double Booking

When attendees attempted to double-book a meeting that was already confirmed in VEMS but had not yet refreshed to display on RoomWizard devices, the system allowed the double-booking and failed to post a conflict status.

Fix: Changed how RoomWizard device-booked events update the database so that double-booked requests post properly in conflict status.
