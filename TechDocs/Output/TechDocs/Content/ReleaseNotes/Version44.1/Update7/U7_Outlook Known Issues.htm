EMS Outlook Known Issues
V44.1, Update 7

Issue

Reference

Request Only Rooms Shown As Restricted

Rooms that are labeled with "Restricted" are actually "Request" rooms.  This is an example of remaining MeetingPlanner language that needs to be removed/replaced.

22360

De-Activating Daylight Savings Time Adjustments Caused Booking Error

Users with the "Automatically adjust for Daylight Saving Time" disabled in Windows will not be able to book rooms at the correct time with EMS for Outlook.

22684

Allow Edit UDFs on Reservation Parameter Not Observed

The "Allow edit of UDF" parameter is not being observed.  All UDFs are editable.

22952

Favorites Missing From Room Search After Upgrade From V42

Favorites that were set in V42 or prior will need to be recreated in V44.1.

23394

Error Message Missing When Booking Past Cutoff Time 

When users attempt to search for a room that would break the "Booking Cutoff Time" rule, users are simply not provided a list of rooms.  No error or warning message currently accompanies this result.

23481

PO Number and Billing Reference Information Display Issues

  • Billing and PO editing is temperamental, with one of the fields not always appearing when it should - during the edit process.
  • There are issues displaying the correct fields in the Edit Billing Information window.
  • Billing information is not being validated when it is edited.  If the user inputs an incorrect reference number, the new number is not saved and the user is not prompted of this action.

23523, 
23524, 
23526

UDF Dropdown Not Scrolling

Large multi-select UDFs and/or having a large number of UDFs available in Outlook will cause the search results section of the add-in to shrink (to accommodate the UDFs).  This can make it difficult to scroll through results, though the user can manually expand the window to gain more space.

23605

Changing Exchange Integration URL Caused Error

Changing the Exchange Integration URL from an http to an https address will not update the appropriate fields in the configuration file.  A manual edit or re-install of the software is currently needed to make this change.  (Changing the address from a "https://test..." to "https://prod..." will work correctly.)

23706

Allow Edit Of Services Parameter Not Observed

"Allow Edit of Services" is not being observed in EMS for Outlook.  All services are editable.

23911

Event Forwarding Failed

If an attendee to a meeting has the Outlook add-in, they will not be able to forward the meeting invite to other users.

24151

Slow Performance Switching From Inbox to Calendar

There are certain configurations in which the add-in will cause Outlook to respond slowly to user input.

24173

Default Service Parameter Not Observed

Setting the default Service Type to "(none)" will not be honored in the Outlook add-in. A default, the first available, will be defaulted for the user automatically.

24324

Error Messages Missing When Changing Reservation With Services To Past Date

When dragging/dropping an existing meeting with services to the past, in violation of the template rules, the service is being cancelled as opposed to being changed to the "Rule Violation" state.

24805

EMS Web App Edits Incorrectly Change Reservation Timezone 

Editing a meeting from the Web App is causing the Outlook appointment to be moved to the wrong time (essentially, the wrong timezone is being used).

24991

"Only Allow New Bookings Prior to This Date" Not Observed

"Only the "...number of days" setting is being observed.

25086

Cancelation Cutoff Time Not Observed The "Cancelation Cutoff Time" is not being observed in EMS for Outlook

The "Cancellation Cutoff Time" is not being observed in EMS for Outlook.

25098

Kiosk Cancelation Not Shown in Outlook

When a meeting is canceled in Kiosk, it is not canceled in Outlook. The add-in is not handling this case well and users trying to access this canceled reservation will encounter an error message.

25178

Updated Location During Conflict Resolution Not Saved 

The Location field in the Outlook appointment is not always being updated appropriately when a conflict is resolved.

25218

Web Bookings Not Shown in Outlook     

Templates that are enabled for Exchange Integration, but not enabled for the Outlook add-in will cause issues in Outlook if a meeting is booked elsewhere and added to the calendar.  The user will encounter an error if they attempt to open the reservation using the add-in. This is considered a bad configuration, but needs to be handled better in the future.

25257
