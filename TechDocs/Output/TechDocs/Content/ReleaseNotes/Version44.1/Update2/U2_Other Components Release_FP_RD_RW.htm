Enhancements 

Area

Description

Floor Plan Configuration Utility

Build #: 44.1.009.100

Bug 21608

Floor Plan Configuration Utility Updated to No Longer User Flash

This utility has been updated to no longer use Flash.

EMS Room Wizard > Start Early

Bug 18797

 

Start Early Option Now Checks Users In To Meetings

The Start Early option in EMS Room Wizard now registers a check-in and appears per the configuration of the device used (minutes prior to start time). No Shows configured for the location do not register if Start Early is checked. 

Note: This only works on the latest Room Wizard firmware version as Start Early did not exist on older versions.

Fix

Area

Description

Room Diagramming Service

Build #: 44.1.0.144

Bug 22380

 

Room Diagramming Service Installed Under DefaultAppPool Regardless of .NET Version

Room Diagramming Service installed under DefaultAppPool by default; if the DefaultAppPool was running .NET 2.0, the service failed because it needed .NET 4.0 (4.5).

Fix: Modified functionality so that the Room Diagramming Service now installs under the EMSAppPool, which is .NET 4.0 (4.5).
