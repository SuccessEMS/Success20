Area

Description

REST

EMS Platform Services now supports CRUD Operations for Everyday user Personalization Settings

Added API to GET, PUT and DELETE everyday user personalization settings with call /webusers/{id}/options and /webusers/{id}/options/{templateid}

Added "ShowAttendeesInScheduleView" field to the existing call /myuser/personalizations

Internal Reference # 33724
(For EMS Internal Use Only)

Buildings that have not Active Rooms Are Excluded From the /webtemplates/{id} REST End Point

If there are buildings that have no active rooms applicable for the web process templates, those buildings will not be returned when calling /webtemplates/{id} Rest End Point
