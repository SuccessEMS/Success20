EMS Master Calendar
Release Notes
V44.1 Update 8

This section describes updates to your EMS Master Calendar.  

See Also: EMS Known Issues

 EMS Master CalendarRelease Notes V44.1 Update 8 and Updating to the Latest Version of EMS (Patches and Builds).

Name: 
V44.1 Update 8

Release Date: 
January 31, 2017

Patch:  
2016.12.12.0

Build #:  
Master Calendar 7 Patch: 7.0.0.094 2016.12.16.0

Master Calendar 8 Patch: 8.0.0.105 2016.12.14.0

Fix

Area

Description

Images

Alt Text Field Missing For Uploaded Images

No Alt text displayed for images associated with events.

Fix: Added a new field so users can now add Alt text when uploading or editing the featured image for a reservation or a booking, as well as for event details. Events that occur in the past (prior to this update) will continue to use the image name or event title.

Note: Version 8 requires a build and a patch. Version 7 only requires the build.

EMS Internal Reference # 23933
(For Internal Use Only)

Display

Custom Theme Colors Not Observed For Admin Panel Headings

Changing the font color by selecting a theme did not change the font color for drop-down headers on the Admin panel.

Fix: Modified the display to observe the custom color settings in all areas.

EMS Internal Reference # 19763
(For Internal Use Only)

Locations Not Listed Alphabetically on Main Page and Manage Events

Locations were not listed alphabetically in the drop-down list for some customers on the main Master Calendar page and on the Manage Events page.

Fix: Modified the display to sort Locations alphabetically.

EMS Internal Reference # 21305
(For Internal Use Only)

Event Tooltip Missing Large Icon

When users moused over events on the calendar grid, the tooltip did not show the large icon.

Fix: Modified the display to show the large event icon.

EMS Internal Reference # 19836
(For Internal Use Only)

Notifications

Notifications Missing Event Location

Email reminder notifications did not include the event location.

Fix: Modified functionality to ensure that notifications include event location. Requires both build and patch.

EMS Internal Reference # 24540
(For Internal Use Only)

Submit an Event

Calendar Popup Persists for End By Field

When users entered a new event and clicked the End By field, the date picker popup continued to display after they clicked a date.

Fix: Modified the popup to automatically close once a user picks a date.

EMS Internal Reference # 23513
(For Internal Use Only)

Events With Multiple Dates Missing Details in API Call

When users entered a new event that had multiple dates, the API call, AddEventWithMultipleDates only returned the event ID, but did not return the event details.

Fix: Modified the API to return event details.

EMS Internal Reference # 25336
(For Internal Use Only)

Featured Events

Featured Events Section Empty

When users resized the Master Calendar window and there were no featured events, this empty section continued to display when it should have been hidden.

Fix: Modified the resize display to hide the Featured Events section when there are none.

EMS Internal Reference # 24024
(For Internal Use Only)

Event Tags

Event Tags Over 100 Characters Caused SQL Error

When users entered more than 100 characters in the Event Tags field and saved the event, an error occurred.

Fix: Modified validations to truncate long tags so the error no longer occurs. 

EMS Internal Reference # 14097
(For Internal Use Only)

Google Maps

Geocode Details Missing

When users provided a Google Maps location URL to retrieve a geocode, details such as latitude, longitude, hours, and phone number were missing because the Google Maps API now requires an additional authentication key.

Fix: Added the authentication key to web.config to ensure that all geocode details are retrieved and displayed for Google Maps URLs: 
<add key="GoogleMapKey" value="AIzaSyALWSXqWl2Kj9KfOxt7TK_QnaovBtzrma8" />.

EMS Internal Reference # 25281
(For Internal Use Only)

Social Networking

Special Characters Displayed Incorrectly In AddThis URLs

When users shared events using the AddThis option, special characters caused URLs to display incorrectly.

Fix: Modified URLs to properly handle special characters.

EMS Internal Reference # 23934
(For Internal Use Only)

Mobile Devices

Calendar Manager Text Cut Off On Mobile Devices

When users clicked the Calendar Manager link when running Master Calendar on a smartphone, text was cut off on the left side

Fix: Modified the display so text is not cut off on mobile devices.

EMS Internal Reference # 24284
(For Internal Use Only)

Calendars

Private Events Incorrectly Shown to Unauthorized Users on Related Events

Featured Events Label Not Shown On Calendar Default Settings

When Administrators set a new label for the Featured Feed on Admin > Site Administration > Default Settings (Labels: Featured Feed Label), the new label was not displayed on the Calendar URLs page (Admin Panel > Admin > Groupings & Calendars > Calendar URLs)

Fix: Modified the display to honor the label setting.

EMS Internal Reference # 23208
(For Internal Use Only)

RSS

Keyword Filters Block RSS Feeds

When users set RSS feeds to search by keyword, nothing was returned.

Fix: Modified RSS feeds to search and return content by keyword.

EMS Internal Reference # 21726
(For Internal Use Only)

Outlook

Overnight Events Showed as Recurrences in Outlook

Events that extended overnight that were imported into Outlook via iCal displayed as recurrences.

Fix: Modified functionality so that overnight events now display in Outlook as single events.

EMS Internal Reference # 26450
(For Internal Use Only)

Connectors

Events From Connectors Absent When DB Blocked

When Master Calendar could not connect to the EMS database on an SQL server, events from Master Calendar connectors disappeared.

Fix: Modified functionality to no longer purge previously loaded events when errors occur connecting to the database.

EMS Internal Reference # 13433
(For Internal Use Only)

User Defined Fields

Multiple-Line UDFs Caused Error

When a reservation prompted users to complete multiple-line UDFs in a search that included an option to specify a Calendar, and error occurred and bookings for that calendar could not be retrieved from the database.

Fix: Modified how user's entries in the UDFs are processed through the connector to ensure that this type of UDF configuration does not cause an error.

EMS Internal Reference # 22932
(For Internal Use Only)
