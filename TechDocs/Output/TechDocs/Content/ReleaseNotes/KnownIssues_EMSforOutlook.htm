EMS Internal Reference

Users cannot book any room if template's "Only Allow New Bookings Within Range of This Many Days," parameter is set to "max value-999999."

33052

Occurrence will have empty "Selected Rooms" tab if user has chosen a room from another building in conflict modal and reservation was created using a video conference (VC) or usual template.

33118

If an invoice has been run for bookings in the future, those bookings can still be edited via EMS for Outlook. Best practice is to only invoice past bookings.
