Booking a Recurrent Meeting

A recurrent meeting calls for rooms that are available for a particular time on multiple days.

  1. To begin, open Microsoft Outlook and set up a standard recurring meeting (using the Recurrence button from the Outlook toolbar); in Outlook, specify:
  • subject
  • dates and times
  • attendees

Tip: Be sure to set an end date for the recurrence. EMS for Outlook does not support infinite recurring meetings.

  1. Click the EMS for Outlook icon in the Outlook toolbar. The EMS for Outlook add-in opens in the List view. The List view shows the availability for up to 750 rooms. The Start Time and the End Time are pulled from the initial meeting your started in Outlook. You cannot edit these values.
  1. On the Select Template field, select a booking template for the type of reservation you are making. Your Administrator may have designated a special template for recurring reservations. The rooms listed as available in the Schedule view change to match the template you select.
  2. On the Event Type field, select the event type.
  3. Use the Groups field to specify the person who is hosting the reservation. Click the Search icon to search by name. See Also: Inviting Groups.

Tip: You can view and manipulate the List view and narrow your room search results. This view is the best way for you to choose from only rooms that are available for this booking (this prevents you from accidentally choosing rooms that might cause a conflict). You can also switch to the Schedule tab at any time to see your booking progress, attendees' availability, and other bookings on the calendar.

Tip: You can add multiple rooms for a single booking. Additionally, if you select a room that does not have all the days available that you requested (the Days Available is less than your total days), a Resolve Conflicts popup will appear where you can select a different room for the unresolved days.

  1. Select the room(s). The Selected Rooms tab shows the number of rooms selected for the meeting.
  2. If you are adding services, See Also: Adding Services.
  3. Click Save and Send. The selected room is booked and automatically added to your Outlook calendar, even if the selected room was not available for all the dates. Regardless of the meeting room availability, all the invited meeting attendees receive a standard Outlook invitation. The invitees accept or decline the meeting invitation as they normally would in Outlook. The next time you open the EMS for Outlook add-in and show the date of the scheduled event, the room status is indicated as "Booked" for the event dates for which the room is available, and "Booking in Conflict" for the event dates for which the room is unavailable.

Important: If your booking is in conflict, see Resolving Booking Conflict.
