System Requirements and Prerequisites

In order to properly install EMS Plan a Meeting (PAM), the following prerequisites and requirements must be met:

  1. EMS must be installed and operational
  2. Virtual EMS (VEMS) must be installed
  3. You must have a valid PAM license

You must be licensed for EMS, Virtual EMS, and PAM in order to configure and use this module. If you are unsure if your organization is licensed for PAM, or if you would like to learn more about it, please contact your Account Executive.

Important:  Plan-a-Meeting configuration issues are often relate to access rights with the Web Server account. Please ensure that the account has the necessary permissions.
