EMS Campus Web Service

Operating System

Windows Server 2008/2008 R2*

When installing the EMS Campus Web Service on a Windows Server 2008/2008 R2 operating system, please ensure the following Web Server (IIS) Role Services are installed:

  • Common HTTP Features
  • ASP.NET
  • IIS 6 Management Compatibility

Note: The EMS Campus Planning Interface also requires Microsoft .NET Framework 4.6.1 on the web server. In Windows Server 2008/2008 R2, this is a feature that can be enabled in Windows Server Manager.
