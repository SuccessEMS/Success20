SAML 2.0 Troubleshooting

This section provides troubleshooting tips and information about SAML. For an overview of SAML terminology and how SAML works, see SAML 2.0 Overview. For instructions on configuring SAML, see .

Troubleshooting Overview

To troubleshoot SAML configuration issues, do the following:

  1. Determine where in the SAML authentication process the error or issue is occurring. Refer to SAML Authentication Process for a diagram and detailed description of the process.
  2. Get a copy of the SAML request (or SAML response if the authentication process has progressed far enough).
  3. Review the SAML request/response to locate potential issue.
