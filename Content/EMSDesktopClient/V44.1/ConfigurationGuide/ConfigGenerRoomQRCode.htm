Configure and Generate Room QR Codes

In order to associate rooms with QR Codes (in EMS Desktop Client), you will: 

  • Enable Room Cards for Rooms
  • Run the Room Card - QR Code Report to Generate QR Codes

This automatically generates the codes and associates them with the designated rooms.

Enable Room Cards for Rooms 

  1. On the menu bar, click Configuration > Facilities > Rooms. The Rooms Configuration page opens.
  2. Select a room for which you want to create a QR code. (For complete instructions on configuring rooms, see Configuring Rooms.) Click Edit.
  3. In the No. Room Cards field, insert a value (a number greater than 1 will allow you to generate a QR code for this room).

    No. Room Cards Field

  4. Click OK to save your changes.

Run the Room Card - QR Code Report to Generate QR Codes 

  1. Select Reports > Hoteling > Room Card - QR Code.

    Room Card - QR Code Selection

  2. In the Room Card- QR Code setup window that appears, use the Buildings, Room Types, and Rooms tabs to specify the parameters for the QR Code report. Use the arrow keys to move your preferences to the selected list. You must specify preferences on all three tabs.

    Specify Preferences on Buildings, Room Types, and Rooms Tabs

  3. You can add a logo and control the order and titles on the report on the Options tab.

Print Logo Selections on Options Tab

 

  • Click Sort Room Cards By Book Sequence to order the QR Room Cards the way they display in the Reservation Book (instead of by Room Name). Book Sequence is set in Room Configuration.
  • To print separate QR Room Cards for each component (titled by Room Name), click Print Combo Room Components.

Tip: We recommend that you use one method to generate multiple cards for the same room: either configure the room to have more than one room card (see Step 1) or set Copies To Print (above) to print multiple copies.

  1. When you have specified all settings for your QR codes, you can click Print Preview to see a preview of the report (ideal for emailing, exporting, printing, and so on), or Print to print the file. 

Sample Print Preview

Warning: If you see the error message below, return to the initial step above. The No. Room Cards field on the Room configuration should be greater than zero.
