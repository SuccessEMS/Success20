Import Resources for a Category

You can import resources for a category from a tab-delimited file (see Resource Import File Specifications for the import file specifications). After you import the resources, you can edit their configurations as necessary.

To import resources for a category, complete the following steps:

  1. On the EMS menu bar, click Configuration > Resources. A list of all the categories that are currently configured in your EMS database and for which you can configure resources opens.
  2. Select the category for which you are importing the resources. The Resources window opens. This window lists all the resources that are currently configured in your EMS database for the selected category and that have a status of Active.

Resources Window

  1. Click the Import button on the Resources window,. The Import Resources dialog box opens.

Import Resources Dialog Box

  1. Click the Search icon to open the Find Resource Import File dialog box, and then browse to and select the resource file that is to be imported (see Resource Import File Specifications for the import file specifications).
  2. From the Rate Schedule drop-down list, select the rate schedule that applies to the resource. (See Also: Configure Rate Schedules.)
  3. Click Import. A dialog box opens, indicating the number of resources that you are about to import and asking you if you want to continue.
  4. Click Yes. A message appears indicating that the import was successful.
  5. Click OK to close the message and return to the Resources window.
