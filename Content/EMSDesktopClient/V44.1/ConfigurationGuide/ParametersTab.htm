Parameters Tab

  1. Open the Parameters tab.

Parameters Tab

  1. By default, an Everyday User Process template inherits its parameter settings from the global list of EMS Web App parameters. You can review the values for these parameters, and edit the values as needed for the template.
    • To edit the value for a parameter, select the parameter, and then click Edit.
    • To reset the value for a parameter to its default global value, select the parameter, and then click Use Global.
  2. Continue with any other configuration for the templates as needed; otherwise, click OK to save the changes to the template. For an Everyday User to be able to view and use this template, you must assign the user to the template. See Assign Templates to Everyday Users.
