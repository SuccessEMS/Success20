Note: In Update 30, the EMS Room Sign API was moved into the application to improve performance. If you have upgraded to Update 30 or later, you do NOT need to install the API or create an EMS user account with EMS API web service access (see prerequisites for Update 30 and later).

Prior to Update 30

If you are running a software version prior to Update 30, you will need to meet the following prerequisites:

  • Install and configure the EMS Desktop Client.
  • Uninstall all previous versions of the EMS Room Sign App web application.
  • Install the EMS Room Sign App API. The EMS Room Sign App API URL is required for the EMS Room Sign App web application installation.
  • Create an EMS user account with EMS API Web Service Access enabled. The user account is necessary to access the database.

Update 30 and Later

If you are running Update 30 or later of the software, you will need to meet the following prerequisites:

  • Install and configure the EMS Desktop Client.
  • Uninstall all previous versions of the EMS Room Sign App web application.
