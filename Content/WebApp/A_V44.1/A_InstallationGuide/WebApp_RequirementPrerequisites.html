Prerequisites

  1. Before beginning the installation process, please install or upgrade your EMS databases as outlined in the EMS Web App (V44.1) Installation Guide.
  2. Installation will prompt you to enter a web server. Special setup guides are available for Windows Server 2008/2008 R2 Web Server Setup Guide and Windows Server 2012/2012 R2 Web Server Setup Guide.
