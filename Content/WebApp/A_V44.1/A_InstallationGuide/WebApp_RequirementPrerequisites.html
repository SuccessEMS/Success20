Prerequisites

IMPORTANT: Beginning with Update 31, EMS Platform Services is a pre-requisite for installing and using the EMS Web App. Please validate hardware and software requirements for EMS Platform Services if it is has not yet been installed in your environment. (If your organization uses the Mobile App, Kiosk, Google Integration, or most partner integrations, then Platform Services is already running in your environment.)

  1. Before beginning the installation process, please install or upgrade your EMS databases as outlined in the EMS Web App (V44.1) Installation Guide.
  2. Installation will prompt you to enter a web server. Special setup guides are available for Windows Server 2008/2008 R2 Web Server Setup Guide and Windows Server 2012/2012 R2 Web Server Setup Guide.  
  3. If using Update 31 or above, install EMS Platform Services.
