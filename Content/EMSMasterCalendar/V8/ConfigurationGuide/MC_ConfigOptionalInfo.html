Configuring Optional Information

In Master Calendar, when you manage the information, you can manage it at a basic level or you can enhance it with optional information. For example, when you are adding a calendar URL, you can add it based solely on event keywords, or you can add it based on optional information such as the event type, the event location, as well as the event keywords. All optional information is maintained in list views and follows a similar user interface design.
