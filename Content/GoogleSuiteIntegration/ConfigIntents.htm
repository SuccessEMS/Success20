Configure Intents in the EMS Desktop Client

Booking Types are configured as Intents in the EMS Desktop Client. Booking Types are a new configuration available for grouping Everyday User Process templates. Everyday User Process templates can be assigned to only one intent.

Booking Types are configured in the EMS Desktop Client. To configure an intent: 

  1. Navigate to the Everyday User Process Templates tab > Details tab.
  2. Provide an Intent Name (there is a 20-character limit).
  3. Designate a Start Time, End Time, and Default Duration.
  4. Menu Sequence.
  5. Ensure your Everyday User Process template has an assigned Priority Value relative to the other templates linked to the intent.

NOTE: Priority is a required field and two process templates cannot have the same priority value. Priority values ≤ 0 should not be assigned to any process template.

The Intents will now display on your Google calendar.
