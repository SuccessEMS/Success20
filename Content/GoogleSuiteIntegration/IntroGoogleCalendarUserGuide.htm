Introduction to Google Calendar

Google Calendar Integration allows access to GSuite users’ Google Calendars to provide their free/busy statuses within EMS for scheduling meetings. Everyday Users can see attendee availability directly within the EMS Web App or Mobile App alongside space availability information from the EMS database.

Google Calendar Integration is managed through EMS Platform Services. Google user authentication is supported through SAML authentication.

NOTE: Google Calendar integration requires an account with GSuite by Google Cloud. For more information please click here.

This User Guide will provide information that will allow you to:

  • Create a Booking
  • Filter Buildings/Locations
  • Select a Room
  • Create Favorite Rooms
  • Set User Preferences
  • Configure Intents
  • Conflict Resolution
  • Authorize URLs for Subscribing

Create a Booking

When creating a booking, users can easily compare room availability alongside invitee availability to find a day and time that works for the right people and in the right space. Additionally, Google Calendar Integration will send invitations to attendee’s Gmail and Calendars.

  1. Create a new calendar item.
  2. Add the name of your meeting to the Add Title field.

    NOTE: A Title is not required; however, if one is not assigned, EMS will provide a default Title based on the intent.

  3. Provide a date and time, ensuring you are in the correct time zone.
  4. From the EMS extension, select an Intent from the drop-down list.

  1. Search for a Room. Use the Filters link to filter by Building or search by Room Name.
  2. Depending on whether the Room is classified as a reserved room, options will include either Book or Reserve. Select a room by clicking either Book or Reserve.
  3. Once a room is selected, a guest mailbox is added to the Guests tab.

NOTE: To remove a Room, click Remove.

Create Favorite Rooms

To create favorite locations in Google Calendar, simply click the heart icon next to the room in the EMS extension.

Set User Preferences

Setting user preferences allows users to designate favorite locations, floors, and rooms. This will expedite the booking process for future reservations.
