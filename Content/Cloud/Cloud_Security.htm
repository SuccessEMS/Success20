Cloud Services Security

Top-Tier Providers and Security

  • Hosted on Google Cloud, Amazon Web Services, and Microsoft Azure
  • Facilities hold ISO27001, ISO27017, ISO27018 certifications
  • Audited to SSAE 16/ ISAE 3402 Type II, SOC 1, SOC 2, SOC 3
  • Data-in-Transit (using HTTPS/TLS) and Data-at-Rest using AES 256-bit encryption
  • Recovery Point Objective (RPO): 15 minutes

Data Security

Cloud Services for EMS provides the following data security:

  • Data-in-Transit always encrypted
    • End user applications to Hosting – SSL / TLS
    • Full encryption between customer and EMS
  • Least Privilege Access granted with principle of least privilege
    • Only EMS Cloud Operations can access your environment
  • Hardening of both application and database servers
  • Monitoring of log files, audit logs, temp files, caches, etc.
