Prerequisites

  1. Uninstall any older versions of EMS for Outlook.
  2. The EMS Integration to Exchange Web Service must be installed and operational. For information on how to install and configure this component.

TIP: You can quickly verify if the service has been installed by opening a browser and entering the following:
http://[ServerName]/ExchangeIntegrationWebService/Service.asmx (replace [ServerName] with the name of your web server)

The Web Service Address will be required when running the EMSForOutlook.msi (described in Where to See Your Exchange Server URL and EMS for Outlook Version Number).

  1. EMS must be configured properly in order to activate EMS for Outlook for each Outlook® user. 
  2. Verify that the required software is installed on your users’ workstations (See Also: EMS for Outlook Requirements).
