Labels

 

Label Changes Were Not Honored

Despite the event label parameter being changed in the EMS Desktop Client, the EMS Mobile App was not reflecting this new label.

Fix: Modified functionality so that label changes are reflected in the EMS Mobile App.

Internal Reference # 33231
(For EMS Internal Use Only)

Process Templates

 

EMS Mobile App-Enabled Process Templates Assigned by LDAP Groups Were Not Assigned to Everyday User When Authenticating via LDAP

When utilizing LDAP Authentication on the EMS Web App and assigning process templates based on LDAP Groups, EMS Mobile App-enabled templates were not assigned to the User when utilizing the EMS Mobile App.

Fix: Modified functionality so that EMS Mobile App-enabled templates assigned via LDAP groups are available on the EMS Mobile App.
