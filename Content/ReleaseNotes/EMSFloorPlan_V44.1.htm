Configuration Tool

Floor Plan Configuration Tool Limiting Number of Indicators

The Floor Plans configuration tool was not allowing more than a certain numbers of indicators to be added to a Floor Plan. The window that assigns the indicators froze and would not allow more saves.

Floor Map Filters

Room Type Filter Issues

When users filtered a floor map indicator by Room Type, the filter was not saved for the next time. Additionally, incorrect room types displayed that were not associated with the selected floor.

Indicators

Indicators Shift Slightly When Saved

When users placed floormap indicators and then saved, the indicators shifted in position slightly.

Incorrect Prompt For Floor When Already Specified

When users selected a floor and then on the floormap image, placed floormap indicators and then saved, they were prompted to specify a floor.

Area

Description

Room Diagramming Service

 

Room Diagramming Service Installed Under DefaultAppPool Regardless of .NET Version

Room Diagramming Service installed under DefaultAppPool by default; if the DefaultAppPool was running .NET 2.0, the service failed because it needed .NET 4.0 (4.5).

Fix: Modified functionality so that the Room Diagramming Service now installs under the EMSAppPool, which is .NET 4.0 (4.5).
