Provisioning

 

Unable to Provision EMS Kiosk App Under Certain Profiles Due to Amount of Rooms

When attempting to choose a profile, the profile modal page in the EMS Kiosk App kept spinning until it timed out if the profile contained more than 1,000 rooms.

Fix: Modified functionality so that users can select a profile regardless of the number of rooms.

Internal Reference # 42979
(For EMS Internal Use Only)

Double Requests on Provisioning and Reauthentication

The EMS Kiosk App was experiencing double requests during provisioning and reauthentication.

Fix: Modified functionality so that the EMS Kiosk App provisions correctly.

Internal Reference # 43014
(For EMS Internal Use Only)
