User-Defined Fields (UDFs)

Unable to Add Previously Existing UDF to Reservation

User-defined fields (UDFs) could no longer be added to a reservation more than once.

Fix: Modified functionality so that users can add a UDF to a reservation more than once.

Internal Reference # 28076
(For EMS Internal Use Only)

Service Order Notification

Selecting Finish on a Service Order Notification Was Resetting State to All

Users who opened an existing Service Order Notification Rule, that includes a Filter for States, but did not make any changes to the rule were experiencing the Finish button reverting the States filter to All. If the user selected the Cancel button, the existing States filter selection was retained.

Internal Reference # 30708
(For EMS Internal Use Only)

Notifications

Email Notifications Were Not Being Sent

When reservations were created with bookings that met the conditions of the Group Notification rule, group email notifications were not being sent.

Fix: Modified functionality so that Everyday Users receive the group notification.

Internal Reference # 33666
(For EMS Internal Use Only)
