Creating or Updating Reservations through EMS for Outlook Caused Microsoft Outlook to Crash

For some Microsoft Outlook 2010 users, creating or updating reservations through EMS for Outlook caused Microsoft Outlook to crash.

Booking Time Changes Made Via Drag-and-Drop in Microsoft Outlook Calendar Were Not Updating in EMS Desktop Client

When users modified an EMS for Outlook booking's time via drag-and-drop in the Outlook calendar, the booking's time did not update in EMS Desktop Client.

Fix: Modified functionality so that if users modify an EMS for Outlook booking's time via drag-and-drop in the Outlook calendar, the booking's time will update in EMS Desktop Client.
