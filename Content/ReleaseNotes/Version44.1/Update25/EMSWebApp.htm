Browse Events

 

Number of Events on Web Events Daily, Weekly, and Monthly Did Not Match

When Browsing Events, users were seeing an incorrect number of events on the Daily, Weekly, and Monthly views.

Fix: Modified functionality so that the correct number of events are displayed on a given date.

Internal Reference # 35014
(For EMS Internal Use Only)

Edit Reservation Page

Add Booking Button Was Appearing on Video Conference Template

Users were briefly seeing the Add Booking button on the Edit Reservation page in the EMS Web App.

Fix: Modified functionality so that the Add Booking button does not ever display on a VC template or Calendar-enabled template.

Internal Reference # 36849
(For EMS Internal Use Only)

LDAP Authentication

 

Intermittent Slowness Experienced During Authentication (EMS Cloud Services Customers)

EMS Cloud Services customers experienced intermittent slowness when authenticating with LDAP.

Fix: Modified functionality to eliminate any latency during LDAP authentication for EMS Cloud Services customers.

Internal Reference # 38324
