Description

Set Course Preferences

Requires Different Room Checkbox Was Not Being Honored

For cross-listed courses, when users changed a room on the parent course, all of the child courses were updated with the new room location even though the Requires Different Room checkbox was selected for the child courses.

Fix: Modified functionality so that when the Requires Different Room checkbox is selected for child courses, the child course locations are not updated when the parent course location is updated.
