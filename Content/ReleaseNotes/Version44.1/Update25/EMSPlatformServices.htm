Enhancements

Area

Description

Search

 

Search Enhancement

Users can now search Buildings by any user-defined field (UDF).

Internal Reference # 37244
(For EMS Internal Use Only)

User-Defined Fields (UDFs)

 

View UDFs for Multiple Rooms

Users can now view UDFs for multiple rooms IDs.

EMS Internal Reference # 37245
(For EMS Internal Use Only)

ParentID

ParentID for Attachments and Drawings

A ParentID can now be used to associate a particular drawing/attachment with a booking.

EMS Internal Reference # 37928
(For EMS Internal Use Only)

Add Auth Key

Dates Detected From the Certificate

Instead of dates being detected from the Add Auth Key Route, they now come from the certificates themselves.

EMS Internal Reference # 38096
(For EMS Internal Use Only)

Database

Database Enhancement to Accommodate 4096-bit Keys

The database can now fit 4096-bit RSA keys.

EMS Internal Reference # 38100
(For EMS Internal Use Only)

GET Statuses

 

Enhanced GET Statuses Functionality

Additional filters have been added so that GET/Statuses returns all statuses, instead of active statuses and those that have the display on web flag set to true.

Fixes

Area

Description
 

Bookings and Reservations Were Created Using Invalid Group ID

Users could create bookings and reservations using invalid data (e.g., Group ID).

Fix: Modified functionality so that reservations and bookings cannot be created if invalid data was passed in the create requests.

EMS Internal Reference # 37549
(For EMS Internal Use Only)
