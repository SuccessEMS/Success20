EMS Room Sign App Running on iOS 11 Devices and ArmourActive Enclosures Not Syncing

Users are required to interact/touch the EMS Room Sign App for it to start playing sounds. This causes LEDs on the ArmourActive enclosures to be out of sync or display a blue light when the EMS Room Sign App (web page) is loading. This issue is the result of Apple making a change to the Web Audio API in iOS 11. It now requires sounds to be triggered from an explicit user action, such as a tap. Calling noteOn() from an onload event will no longer play sound. Please find more information from Apple here.
