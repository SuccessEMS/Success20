Certificates

SP and IdP Certificates Stored in EMS Database

EMS Platform Services stores multiple copies of the IdP public key, SP public key, and SP private key on the database.

Internal Reference # 33860
(For EMS Internal Use Only)

Security

SP Private Certificate Encrypted Using Key

EMS Platform Services provides enhanced encryption by using a private key that is encrypted before being stored. There is a default encryption key for customers who do not want to manage encryption keys.

EMS Internal Reference # 34134
(For EMS Internal Use Only)

Due to an installer certificate issue, the web deployment of EMS Desktop Client was blocked.
