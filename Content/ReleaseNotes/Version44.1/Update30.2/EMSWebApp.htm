Skype for Business

 

 

Incorrect Formatting on Skype for Business Invite Generated by EMS Web App

Footer text was not displaying correctly when using certain characters (" < >) and Help link were missing on invitations generated by the EMS Web App.

Fix: Modified functionality so that special characters are properly displayed and custom footer can be utilized.

Browse Locations

Browse Locations Page Loading Slowly

Browse Locations was slow upon initial loading of the Schedule Grid.

Fix: Updated the webapp_Common_GetBookEvents stored procedure to improve load times in Browse Locations page.

Internal Reference # 41636
(For EMS Internal Use Only)
 

Incorrect Time Displayed on the Schedule in Locations

For Web Process templates with Get Availability enabled, event timelines for locations were incorrect and no available rooms were displayed.

Fix: Modified functionality so that timelines for locations display correctly.
