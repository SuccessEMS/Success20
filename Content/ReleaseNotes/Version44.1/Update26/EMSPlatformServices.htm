API Calls

Edit In-Progress Bookings

Modified the PATCH/bookings/{id} route to allow editing the Start Time and End Time fields for a booking that is in-progress:

Note that in-progress edits will NOT be supported on the following booking types:

  • Bookings with service orders
  • Bookings with setup/teardown times
  • Video conference bookings
  • Integrated calendar bookings (Microsoft Exchange and G Suite)
  • Managed space (request rooms) bookings

For this enhancement, no changes were made to the notifications that are triggered on changing the start time and end time.

EMS Kiosk >
Apply Profile

Unable to Apply EMS Kiosk Profiles

EMS Kiosk profiles were being created, but users could not apply the profile. Users could not get past the "Are you sure you want to apply this profile" screen.

Fix: Modified functionality so that users can apply EMS Kiosk profiles.

EMS Internal Reference # 38779
(For EMS Internal Use Only)

EMS Mobile App

EMS Mobile App Was Not Displaying Bookings on the Home Page After the Latest EMS Platform Services API Build

The EMS Mobile app was not showing bookings on the Home page after the latest EMS Platform Services API build.

Fix: Modified functionality so that the EMS Mobile app displays bookings on the Home page.

EMS Internal Reference # 39359
(For EMS Internal Use Only)

EMS for Outlook
API Calls

Unable to Change Room from "None" or "TDB" to Another Room Type

When trying to change a room from "None" or "TBD" to another room type, a "403 Forbidden" response to the Patch /bookings/{id} request is returned.

Fix: Modified functionality so that a "200 OK" response to the Patch /bookings/{id} request is returned and the room is changed.

EMS Internal Reference # 39814
(For EMS Internal Use Only)

EMS for Outlook

API Calls

EMS for Outlook Add-in Did Not Show the Floor Filter

The GET webtemplates/{id} returns showFloors=false even though the Show Floor Filter template in EMS Desktop Client has the parameter value set to true.

Fix: Modified functionality so that showFloors takes its value from the Show Floor Filter template parameter and the EMS for Outlook Add-in can display the floor filter.

EMS Internal Reference # 39877
(For EMS Internal Use Only)
