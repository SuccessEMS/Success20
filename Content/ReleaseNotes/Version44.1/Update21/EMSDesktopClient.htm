FIXes

Area

Description

Reservation Wizard

Reconfirm Date for Status Was Not Available

The reconfirm date field was unavailable on the Reservation Wizard template when a status was selected.

Fix: Modified functionality so that when a status is selected, the reconfirm date field is available to users and must be entered before proceeding.

Internal Reference # 35048

Event Name

Time in .ICS File Did Not Adjust for Time Zone

When booking an event, the time indicated in the .ics calendar file was not being adjusted for the local time zone. It displayed the same hour start and end as the navigator.

Fix: Modified functionality so that the time displayed in the .ics file accounts for the difference in time zone.

Internal Reference # 29342

Time Zone

Missing Time Zone

The Novosibirsk, Russia time zone is missing.

Fix: Modified functionality so that time zone Novosibirsk is available for selection.
