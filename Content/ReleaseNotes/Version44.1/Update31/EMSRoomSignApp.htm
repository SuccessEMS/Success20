Iadea Firmware

Updated Iadea Firmware Tested with EMS Room Sign App

EMS Room Sign App has been tested and verified with the following Iadea firmware: 

  • Firmware 1.2.91.264, Android 4.4
  • Firmware 2.0.0-27, Android 7.1

Important: NFC authentication does not work with Iadea firmware version 2.0.0-27. If you want to use NFC authentication, you will need to install the older firmware version (1.2.91.264, Android 4.4).

Check In

Check In Button Was Being Activated Early

The Check In button was being activated earlier than the time specified in the EMS Room Sign App settings. For example, a room was booked at 9:00 a.m., and the profile settings specified the Check In button would activate 10 minutes before the event start time. However, instead of displaying the Check In button at 8:50 a.m., the Check In button appeared at 8:45 a.m. (reserved start time/setup time).

Fix: Modified functionality so that the Check In button is activated at the time specified in the EMS Room Sign App settings.
