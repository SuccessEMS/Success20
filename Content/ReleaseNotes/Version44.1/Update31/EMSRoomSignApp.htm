Profile Settings

Check In Button and Auto Cancel Not Honoring Profile Settings

For some users, the Check In button would display earlier than the time specified in the profile settings. Additionally, auto cancel would not always run at the time specified in the profile settings.

Fix: Modified functionality so that the Check In button and auto cancel honor the EMS Room Sign App profile settings.

Internal Reference # 41128
(For EMS Internal Use Only)

Licensing
Troubleshooting

Added Documentation for Troubleshooting "Exceeds License Count" Error

EMS Room Sign App documentation was updated with information to help troubleshoot the error message "exceeds license count." This error message sometimes appears when a tablet is moved to a new enclosure.

Internal Reference # 41612
(For EMS Internal Use Only)

Crestron Screen Display

Screen Interface Was Being Cut Off on Crestron Panels

For customers using Crestron panels, portions of the screen interface were being cut off on the following models and resolutions:

  • 7-inch Creston Panel at a resolution of 1024 x 600 (TSS-7-W-S-SIW KIT)
  • 10-inch Crestron Panel at a resolution of 1280 x 800 (TSS-10-B-S-SIW KIT)

Fix: Modified functionality so that the interface displays correctly.
