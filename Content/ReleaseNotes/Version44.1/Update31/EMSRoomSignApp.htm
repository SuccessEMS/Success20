Iadea Firmware

Updated Iadea Firmware Tested with EMS Room Sign App

EMS Room Sign App has been tested and verified with the following Iadea firmware: 

  • Firmware 1.2.91.264, Android 4.4
  • Firmware 2.0.0-27, Android 7.1

Important: NFC authentication does not work with Iadea firmware version 2.0.0-27. If you want to use NFC authentication, you will need to install the older firmware version (1.2.91.264, Android 4.4).
