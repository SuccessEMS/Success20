EMS Mobile App Did Not Honor Template Booking Rules

Users were unable to create a booking past the current day when the setting Allow this client to book without Everyday User Templates and Ignore Booking Rules was disabled in EMS Platform Services (Integrations tab). 

Fix: Modified functionality so that users can create bookings according to the booking rules specified in the template.
