Create or Edit Reservation

Error After Creating a Reservation in the EMS for Outlook Add-in

For some EMS for Outlook add-in users, when they used a template to create a reservation, an error occurred after clicking Save &amp; Close or Send, and the reservation was not saved in the EMS database. However, the invite was sent to attendees.

Fix: Modified functionality so that when users create a reservation in the EMS for Outlook add-in, the template information populates appropriately, and the reservation is created with no errors.

Internal Reference # 36800
(For EMS Internal Use Only)

Combination Rooms

Combo Room Components Showed Incorrectly in EMS for Outlook

When reserving a combo room in the EMS for Outlook add-in, the Schedule View and List View showed the combo room as unavailable once the combo room was selected; however, the component rooms were displayed as available (+ symbol showed next to the component rooms).

Fix: Modified functionality so that Schedule View and List View show the component rooms as unavailable if the parent combo room has been selected.

Internal Reference # 39620
(For EMS Internal Use Only)

Recurring Reservation

Send Update Button Not Updating Database

EMS for Outlook was not updating the time in the Database when users dragged/dropped a single occurrence of a recurring reservation and used the EMS for Outlook Send Update button, unless the time was changed in the meeting dropdown. This issue also occurred for single reservations; however, there was no delay using the native Outlook Send Update button.

Fix: Modified functionality so that dragging and dropping alone in EMS for Outlook updates the time in the database.

Internal Reference # 41171
(For EMS Internal Use Only)

Cancellations

Canceling One Occurrence from EMS for Outlook Canceled All Bookings in the EMS Web App and EMS Desktop Client

Clicking the Cancel button for one occurrence in EMS for Outlook canceled all bookings in the EMS Web App and EMS Desktop Client.

Fix: Modified functionality so that the Cancel button in EMS for Outlook cancels only the current occurrence and not all.

Internal Reference # 41533
(For EMS Internal Use Only)
