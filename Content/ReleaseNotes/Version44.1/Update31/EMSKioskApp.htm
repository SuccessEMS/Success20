Area

Description

Make a Reservation

Required Rooms Removed From EUPT Room Still Displayed as Available

When the Enable Everyday User Process Template Security parameter is set to YES and a required room is removed from Available rooms, the room was still displaying as Available.

Fix: Modified functionality so that an EUPT room removed does not display.

Internal Reference # 32352
(For EMS Internal Use Only)

Building Closed Screen Did Not Display Future Closures

If the "Start Date Allowed for New Reservation" was configured for the same amount of days as the "Number of Future Concurrent Days" and the building was closed on that day, rooms were still displaying as available.

Fix: Modified functionality so that the Building Closed screen displays and No Rooms are available.

Internal Reference # 33003
(For EMS Internal Use Only)

Building Closed Screen Appeared if Closed All Day on the Previous Day (Web Only)

The Building Closed screen was appearing if the building was Closed All Day on the day before. This is a Web-only issue.

Fix: Modified functionality so that users are able to search and book spaces.

Internal Reference # 34384
(For EMS Internal Use Only)

On Profiles with 2,000+ Rooms, All Rooms Were Displaying As Unavailable on Floorplan

Newly mapped locations on floorplan displayed with "unavailable" indicators.

Fix: Modified functionality so that room availability on the floorplan corresponds with reservations for the required date.

Internal Reference # 36835
(For EMS Internal Use Only)
 

List View Displayed After Reserving a Room on the Profile with 2,000+ Rooms Added

When creating a new or editing an existing Kiosk profile with 2,000+ rooms, the Room List persisted instead of the Confirmation view being displayed.

Fix: Modified functionality so that the Confirmation view displays after reserving a Room and the Room List disappears.

Internal Reference # 36837
(For EMS Internal Use Only)

Session Timeout

For Already Expired Sessions, Session Timeout Modal Appeared

Session Timeout modal appeared for sessions that had already expired.

Fix: Modified functionality so that Session Timeout modal only appears when session is active.

Internal Reference # 41979
(For EMS Internal Use Only)

Today's Events

Scroll Not Working With Multiple Options

When there were multiple options in the EMS Kiosk App, users were unable to scroll.

Fix: Modified functionality so that users can scroll through multiple options.

Internal Reference # 32567
(For EMS Internal Use Only)

 

Unable to Change Overnight Start/End Times in the Picker a Second Time

When trying to edit overnight start/end times for a second time, users received errors that they cannot select times in the past.

Internal Reference # 34413
(For EMS Internal Use Only)
   

Locate a Space

Component Rooms Displayed as Available When Combo Was Booked

When combo rooms were booked, they still appeared as available on the Locate Space screen.

Fix: Modified functionality so that if a combo room is booked, the component rooms appear as unavailable.

Internal Reference # 33144

Language was reverting to default as soon as the session timeout screen appears, regardless of whether the user chose to continue their session.
