Provisioning Screen

Search Field Added to Provisioning Screen

A search field has been added to the Provisioning Screen that allows an Administrator to search for the desired profile (rather than scrolling). This is especially important for customers adopting the EMS Kiosk App on a large scale.

Authentication

Single Authentication Can Access All Authenticated Functions

Users only need to present authentication credentials once within a session to access areas of the EMS Kiosk App that require authentication.

(For EMS Internal Use Only)
 

 

Fixes

Area

Description

Private Bookings

Private Functionality Not Working

If a booking was in an event type, status, or room that was set 'not to display,' the Event Name of the booking did not appear on the EMS Kiosk App when hovering or displaying event.

Fix: Modified functionality so that the parameter value for "label for private bookings on schedule view" displays.

Internal Reference # 34531
(For EMS Internal Use Only)

External Reference

External Reference Label Not Applying

Users were being prompted for their External Reference to create a reservation.

Fix: Modified functionality so that the External Reference label is applied in the EMS Kiosk App.

Internal Reference # 40259
(For EMS Internal Use Only)

Notifications

Notifications Not Triggering

Notifications were not being triggered to targeted recipient when a booking time changed.

Fix: Modified functionality so that notifications are triggered when a booking time changes.

Internal Reference # 40311
(For EMS Internal Use Only)
