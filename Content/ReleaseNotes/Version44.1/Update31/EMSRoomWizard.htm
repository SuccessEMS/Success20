RoomWizard

EMS RoomWizard Connector Did Not Work If SQL EMSUser Account Password Was Changed

If the SQL EMSUser account password was changed, users would get the error "Could not build list of available data sources" when they ran the EMS RoomWizard connector.

Fix: Modified functionality so that when users modify the SQL EMSUser account password, the EMS RoomWizard connector can connect to the EMS database without generating an error.

Internal Reference # 32326
(For EMS Internal Use Only)

RoomWizard

EMS RoomWizard Connector Did Not Work Without SQL EMSUser Account

If the SQL EMSUser account was modified or replaced with a different account, such as an Active Directly account, users would get the error "Could not build list of available data sources" when they ran the EMS RoomWizard connector.

Fix: Modified functionality so that when users modify or replace the SQL EMSUser account, the EMS RoomWizard connector can connect to the EMS database without generating an error.

Internal Reference # 32326
(For EMS Internal Use Only)
