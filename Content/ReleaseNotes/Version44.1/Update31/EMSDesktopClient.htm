Fixes

Area

Description

Email

Unable to Send Emails Using Gmail SMTP Email Settings

Users were unable to send emails from EMS Desktop Client when using Gmail SMTP email settings. When trying to send an email, they would get the error message:

"The SMTP server requires a secure connection or the client was not authenticated. The server response was: 5.5.1 Authentication Required."

Fix: Modified functionality so that users can send emails from EMS Desktop Client when using Gmail SMTP email settings.

Internal Reference # 41280
(For EMS Internal Use Only)

Modify Booking Details

Modify Booking Details Expert User Template Behavior

Modify Booking Details expert user template process was not behaving as expected. The template was not allowing users to check in visitors or modify the visitor category (e.g., add, edit, or delete visitors).

Fix: Modified functionality so that the Modify Booking Details security process allows expert users to edit booking details on any booking.

Internal Reference # 42117
(For EMS Internal Use Only)
