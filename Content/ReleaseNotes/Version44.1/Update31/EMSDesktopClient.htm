Modify Booking Details

Modify Booking Details Expert User Template Behavior

Modify Booking Details expert user template process is not behaving as expected. It is not allowing users to check in Visitors or modify the Visitor category (e.g., add, edit, or delete Visitors).

Fix: Modified functionality so that the Modify Booking Details security process allows users to edit booking details on any booking.
