Enhancements

Area

Skype for Business

Region-specific Dial-in Information Provided

Skype for Business meeting invitation information generated through the EMS Web App now includes only the dial-in numbers for the applicable region in which the user belongs, along with a link to reach the full dial-in number list.

Internal Reference # 41483
(For EMS Internal Use Only)

'Try Skype Web App' URL and Text Added

The link to 'Try Skype Web App' has been added to the meeting invitation information generated when a meeting is initiated in the EMS Web App.

Internal Reference # 41487
(For EMS Internal Use Only)

Enhanced Formatting

The font, size, and color of Skype for Business information that appears in the meeting invitation when a meeting is initiated from the EMS Web App now matches the native Skype for Business add-in exactly.

Internal Reference # 41490
(For EMS Internal Use Only)

Inclusion of Disclaimer Text if Configured within Skype for Business

If an organization has configured HTML-supported disclaimer text within the Skype for Business application, the meeting invitation generated through the EMS Web App will now contain the disclaimer text. Code indicating meeting invite is a Skype meeting is added to enable “join skype” on video conferencing devices.

Internal Reference # 41544
(For EMS Internal Use Only)

Fixes

Area

Description

G Suite

Unable to Manage Attendees in the EMS Web App Added from GSuite Integration

Once an initial reservation was created, users were unable to add, edit, or remove GSuite attendees from the Manage Attendees screen. An error was presented when trying to save.

Fix: Modified functionality so that users can manage attendees (e.g., add, edit, cancel) from the EMS Web App, without any errors.

Internal Reference # 40955
(For EMS Internal Use Only)

Browse Locations

Browse Locations Page Loading Slowly

Browse Locations was slow upon initial loading of the Schedule Grid.

Fix: Updated the webapp_Common_GetBookEvents stored procedure to improve load times in Browse Locations page.

Internal Reference # 41636
(For EMS Internal Use Only)
 

Incorrect Time Displayed on the Schedule in Locations

For Web Process templates with Get Availability enabled, event timelines for locations were incorrect and no available rooms display.

Fix: Modified functionality so that timelines for locations display correctly.

Internal Reference # 41649
(For EMS Internal Use Only)
