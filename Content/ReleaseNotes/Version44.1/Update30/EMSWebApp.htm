Cancel All Bookings for a Date Simultaneously

Users can now cancel all bookings for a date at the same time on a Video Conference (VC) template. Cancellations no longer have to be processed in two steps to cancel multiple locations for a full date or for a full reservation.

Fixes

Area

Description

G Suite

Unable to Manage Attendees Added from G Suite

Users could not manage attendees in the EMS Web App if those attendees were added from G Suite. After editing Attendee(s), user could not proceed to save the reservation.

Fix: Modified functionality so that users can manage attendees (e.g., add, edit, cancel) from the EMS Web App, without any errors.

Internal Reference # 40955
(For EMS Internal Use Only)

Add / Edit Services

PO Number doesn't get updated when changed on Add/Edit Services form in the Web App

The Post Office (PO) Box number field in the Billing Information tab in the EMS Desktop Client was not being updated when services were edited or added in the EMS Web App.

Fix: Modified functionality the PO number field is updated when services are edited or added in the EMS Web App.
