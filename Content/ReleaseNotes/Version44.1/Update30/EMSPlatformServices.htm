Enhancements

Area

Description

Skype for Business

 

 

 

Show Only Region-Specific Dial-in Information

Instead of displaying all local dial-in numbers, Skype information generated through EMS Web App now only shows dial-in numbers from a specified region (the region is specified within the UCWA response). A URL will also be included for other dial-in numbers (URL is included in the UCWA Response).

EMS Internal Reference # 41481
(For EMS Internal Use Only)

"Trouble Joining" Text and URL Added to Skype for Business Information

The "Trouble joining? Try Skype Web App" text and URL have been added to the Skype for Business information generated through the EMS Web App.

EMS Internal Reference # 41484
(For EMS Internal Use Only)

Enhanced Formatting

The font, size, and color of the Skype for Business information that is generated through EMS Web App now matches the formatting in the native Skype for Business add-in.

EMS Internal Reference # 41486
(For EMS Internal Use Only)

Option for Added Disclaimer Text

Option is now available to add HTML-supported disclaimer text to the Skype for Business information generated through the EMS Web App.

EMS Internal Reference # 41488
(For EMS Internal Use Only)

Auto-Summary

Event Name and Event Type Displayed As "Event" in Auto-Summary

The EMS for Outlook auto-summary displayed both the event name and event type with the label "Event."

Fix: Modified functionality so that the auto-summary displays the labels "Event Name" and "Event Type" instead of just "Event."
