Time Zone

Start Date for Brasilia Time Zone DST Incorrect

The start date for Brasilia daylight saving time (DST) has changed to the first Sunday of November. The database time zone table (tblTimeZone) has been updated with the new DST start date for the Brasilia time zone (ID=53); however, all existing bookings in buildings in the Brasilia time zone also require updating to reflect the new Brasilia DST start date.

Fix: Modified functionality so that the Brasilia time zone DST is appropriately updated in EMS applications and in all bookings with buildings in the Brasilia time zone.

Internal Reference # 41570
(For EMS Internal Use Only)

Performance

Browse Locations Page Loading Slowly

For some customers, the Browse Locations page was taking 55+ seconds to display the room schedule grid.

Fix: Modified functionality so that the Browse Locations page loads more quickly.
