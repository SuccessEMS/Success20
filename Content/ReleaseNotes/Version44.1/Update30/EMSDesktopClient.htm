Audit Tab

Updating a Resource Changed the Audit Tab on Everyday User Process Templates Tied to that Category

The Audit tab was updating the Everyday User Process templates after resources in that category were updated.

Fix: Modified functionality so that the Audit tab updates when changes are made in the Everyday User Process template.

Internal Reference # 41113
(For EMS Internal Use Only)

EMS Dashboard

 

Notification Timezone Issues

Reservation notifications on the EMS Dashboard were reflecting the server timezone, and not the timezone where the user created/was viewing the reservation.

Fix: Modified functionality so that the timezone displayed on the EMS Dashboard is the timezone where the user created/viewed the reservation.

Internal Reference # 30419
(For EMS Internal Use Only)

Audit Time Stamps Reflected Incorrect Time

Audit time stamps in the EMS Dashboard and Manage Services were reflecting the local time of the SQL server, instead of the local time of the user.

Fix: Modified functionality so that the dateTime should reflect the time set on the user's machine.

Internal Reference # 34510
(For EMS Internal Use Only)

Authentication

LDAP Authentication Failed if Advanced Communication Enabled

Authentication to the EMS Desktop Client using LDAP did not work when advanced communication options were enabled.

Fix: Modified functionality so that users can authenticate to the EMS Desktop Client using LDAP with or without advanced communications options enabled.

Internal Reference # 41041
(For EMS Internal Use Only)

Modify Booking Details

Modify Booking Details Expert User Template Behavior

Modify Booking Details expert user template process is not behaving as expected. It is not allowing users to check in Visitors or modify the Visitor category (e.g., add, edit, or delete Visitors).

Fix: Modified functionality so that the Modify Booking Details security process allows users to edit booking details on any booking.

Internal Reference # 41308
(For EMS Internal Use Only)
