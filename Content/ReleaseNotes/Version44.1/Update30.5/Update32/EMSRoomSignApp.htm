Fixes

AREA DESCRIPTION
LED 

LED Sync Color Issue

When a user created a reservation in front of an existing meeting that was already allowing check-in, the EMS Room Sign App displayed the check-in LED color and the check-in button while displaying the reserved meeting.

Fix: Modified functionality so that the EMS Room Sign App switches to the reserved LED color until the meeting ends and then switches back to the check-in LED status color while hiding the check-in button for the future event.

Internal Reference # 35825
(For EMS Internal Use Only)

Secondary Authentication

Secondary Authentication Field Not Clearing

When users entered incorrect values in the EMS Room Sign App, the secondary authentication fields were not clearing.

Fix: Modified functionality so that fields can be cleared when incorrect values are entered into the EMS Room Sign App.

Internal Reference # 36612
(For EMS Internal Use Only)
 

ArmorActive Enclosure Was Not in Sync with EMS Room Sign App

The ArmorActive LED light was not in sync with the EMS Room Sign App status after the device polling interval time passed. The EMS Room Sign App was not triggering a sound when the device polling interval time passed.

Fix: Modified functionality so that the EMS Room Sign App triggers a sound when the device polling interval time passes. This results in the enclosure LED light syncing with the EMS Room Sign App status.

Internal Reference # 36960
For EMS Internal Use Only)

Profile Settings

Check In Button and Auto Cancel Not Honoring Profile Settings

For some users, the Check In button would display earlier than the time specified in the profile settings. Additionally, auto cancel would not always run at the time specified in the profile settings.

Fix: Modified functionality so that the Check In button and auto cancel honor the EMS Room Sign App profile settings.

Internal Reference # 41128
(For EMS Internal Use Only)

Crestron Screen Display

Screen Interface Was Being Cut Off on Crestron Panels

For customers using Crestron panels, portions of the screen interface were being cut off on the following models and resolutions:

  • 7-inch Creston Panel at a resolution of 1024 x 600 (TSS-7-W-S-SIW KIT)
  • 10-inch Crestron Panel at a resolution of 1280 x 800 (TSS-10-B-S-SIW KIT)

Fix: Modified functionality so that the interface displays correctly.

Internal Reference # 41953
(For EMS Internal Use Only)
