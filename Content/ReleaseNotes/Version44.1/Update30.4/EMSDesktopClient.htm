Modify Booking Details Expert User Template Behavior

The Modify Booking Details expert user template process was not behaving as expected, as users could not add/edit/delete the category and resources of booking detail items.

Fix: Modified functionality so that the Modify Booking Details security process allows expert users to edit booking details on any booking they have permission to access and edit.
