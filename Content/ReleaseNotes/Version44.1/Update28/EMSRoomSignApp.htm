Area

Checking In

VC Reservation Removed from EMS Web App if Users Checked in Via the EMS Web App

Once EMS Room Sign App users checked into a Video Conference (VC) reservation created in the EMS Web App with check in enabled, the reservation was removed from the EMS Web App. The reservation would still be visible from the EMS Desktop Client.

Fix: Modified functionality so that VC reservations do not disappear from the EMS Web App following check in from the EMS Room Sign App.
