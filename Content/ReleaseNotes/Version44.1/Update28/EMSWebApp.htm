LDAP Authentication

 

Intermittent Slowness Experienced During Authentication (EMS Cloud Services Customers)

EMS Cloud Services customers experienced intermittent slowness when authenticating with LDAP.

Fix: Modified functionality to reduce latency during LDAP authentication for EMS Cloud Services customers.

Internal Reference # 38324
(For EMS Internal Use Only)

Create Events

Items on the Schedule Grid Displayed Improperly

When a booking was created in Indian Standard Time (IST), the red time-indicator in the Schedule Grid in Create Events displayed the incorrect time selection for all buildings and attendee availability. However when the booking was selected, the time displayed correctly.

Fix: Modified functionality so that the correct timezone and attendee availability are displayed in the Schedule Grid.

Internal Reference # 39573
(For EMS Internal Use Only)

Service Orders

VC Booking Resource Incorrectly Duplicated

Users adding multiple VC resources from the same Multi Media category were experiencing duplication of Service Orders being added to their booking.

Fix: Modified functionality so that Services Orders are only added to a booking once and are not duplicated.

Internal Reference # 39952
(For EMS Internal Use Only)
