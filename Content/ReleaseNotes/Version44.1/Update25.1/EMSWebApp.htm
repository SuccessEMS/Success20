Internal Server Error Occurred When Creating a Reservation in the EMS Web App

For customers updating their software to Update 25 from Update 16 or prior, an internal server error occurred when users created a reservation in the EMS Web App.

Fix: Modified functionality so that when customers update their software from Update 16 or prior, users can create a reservation without an internal server error occurring.
