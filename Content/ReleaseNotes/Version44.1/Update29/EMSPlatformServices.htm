Enhancements

Area

Description

SAML Authentication

 

SP Metadata URL Displayed

For the convenience of users configuring SAML authentication, a field has been added to display (but not edit) the metadata URL.

EMS Internal Reference # 38092
(For EMS Internal Use Only)

Metadata Import

The SAML configuration screen allows users to import IdP metadata by file upload and by URL. The file mechanism will use the existing API route to upload; however, the url mechanism needs an alternative route.

EMS Internal Reference # 38093
(For EMS Internal Use Only)

Logs

Enhanced Log Filtering Functionality

Raw and formatted logs can now be filtered by date and time in the EMS Platform Services Admin Portal. This enhancement will facilitate troubleshooting.

EMS Internal Reference # 40234
(For EMS Internal Use Only)

Fixes

Area

Description

GSuite

GSuite Integration Error

Calendaring details were not displayed when viewing Google Calendar meetings from the EMS Web App.

Fix: Modified functionality so that users do not receive errors when integrating with GSuite.

EMS Internal Reference # 40067
(For EMS Internal Use Only)

Edit a Reservation

Room Change Errors in Google Calendar

When changing rooms and date/time in occurrence, the date and time changed, but the room did not.

Fix: Modified functionality so that simultaneous changes to time and location function correctly.

EMS Internal Reference # 38604
(For EMS Internal Use Only)

Skype for Business

Missing Skype for Business Conferencing Details

Users who created reservations in the EMS Web App could not view their Skype for Business conferencing details in Microsoft Outlook.

Fix: Modified functionality so that Skype for Business conferencing details can be viewed in Microsoft Outlook when the reservation is made in the EMS Web App.

EMS Internal Reference # 40765
(For EMS Internal Use Only)

Security

Security Enhancement

A security issue has been resolved. Submit an EMS Support Help Desk ticket through the EMS Customer Support Portal to receive additional information.

EMS Internal Reference # 40971
(For EMS Internal Use Only)
