Edit Reservation

Added Ability to Swap Bookings with Override Description Room and Disregard Conflicts in the Override Room

Users can now swap bookings with override description rooms, and the software will no longer check for availability in the override rooms.

This enhancement gives admins the ability to swap a booking in an override room (for example, Booking A) with a booking in a configured room (for example, Booking B). This can be useful in cases where Booking A has a higher priority than Booking B. Therefore, Booking A is moved to the configured room, and Booking B is moved to the override description room to await reassignment to another configured room.

Area

Description

Web Deploy

Web Deploy Removed Custom DLL Files When Updating EMS Desktop Client

When updating EMS Desktop Client, web deploy would delete the entire application directory and rebuild it. If customers had placed organization-specific DLL files in the application directory (such as files for custom reports, custom badge printing, or EMS Room Sign configuration), these DLL files would be deleted and need to be added back into the application directory each time the web deploy updated the EMS Desktop Client.

Fix: Modified functionality so that when updating EMS Desktop Client, or installing a new version, the web deploy no longer deletes the entire application folder. Instead, it checks, file by file, to ensure custom DLL files remain in the application directory.
