Added Ability to Swap Bookings with Override Description Room and Disregard Conflicts in the Override Room

Users can now swap bookings with override description rooms, and the software will not check for availability in the override rooms.

This enhancement gives admins the ability to swap a booking in an override room (for example, Booking A) with a booking in a configured room (Booking B). This can be useful in cases where Booking A has a higher priority than Booking B. Therefore, Booking A is moved to the configured room, and Booking B is moved to the override description room to await reassignment to another configured room.
