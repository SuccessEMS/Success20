Area

Description

Create a Reservation

Error Message Experienced When Creating Future Meetings

When creating a meeting in the future, users were able to create the reservation but experienced a "Room is not available," error message.

Fix: Modified functionality so that the "Room is booked successfully," message is shown when creating a reservation in the future.

Internal Reference # 36607
(For EMS Internal Use Only)

Security

Security Enhancement

A security issue has been resolved. Submit an EMS Support Help Desk ticket through the EMS Customer Support Portal to receive additional information.

Internal Reference # 40118, 40669
(For EMS Internal Use Only)

Licensing

Removing a Room From a Profile Did Not De-register the Room

Removing a room from a profile was not de-registering the room and was counting against the users' licenses.

Fix: Modified functionality so that removing a room from a profile de-registers the room and no longer counts against the users' licenses.

Internal Reference # 40481
(For EMS Internal Use Only)
