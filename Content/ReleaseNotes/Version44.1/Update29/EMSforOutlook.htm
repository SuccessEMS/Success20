Resolve Conflicts

Resolve Conflict Window Did Not Appear in Microsoft Outlook 2013

In Microsoft Outlook 2013, the Resolve Conflict window did not appear when conflicts occurred for users who were acting as delegates.

Fix: Modified functionality so that the Resolve Conflict window appears when conflicts occur.

Create a Reservation

Combo Room Components Showed Incorrectly in EMS for Outlook

When reserving a combo room in the EMS for Outlook add-in, the Schedule View and List View showed the combo room as unavailable once the combo room was selected; however, the component rooms were displayed as available (+ symbol showed next to the component rooms).

Fix: Modified functionality so that Schedule View and List View show the component rooms as unavailable if the parent combo room has been selected.

Internal Reference # 39620
(For EMS Internal Use Only)

Conflict Resolution and
Recurring Reservations

EMS for Outlook Add-in Was Unresponsive When Users Tried to Resolve Conflicts

When resolving booking conflicts for a recurring meeting, the select room button (+) on the Resolve Conflicts page was unresponsive, and users were unable to select a new location to resolve the conflict.

Fix: Modified functionality so that the select room button (+) on the Resolve Conflicts page functions properly, and users can select a new location to resolve a conflict.

Internal Reference # 40727
(For EMS Internal Use Only)
