Area

Description

ActiveReports
Web Deploy

ActiveReports Library Not Licensed

When users generated a confirmation or report, a message appeared at the bottom of the confirmation/report stating that the document was created using an evaluation version of ActiveReports and that only a licensed user may legally create reports for use in production.

Fix: Modified functionality so that the ActiveReports license is valid, and the invalid license message does not appear.

Internal Reference # 38828
(For EMS Internal Use Only)

Attachments
Web Deploy

Unable to Attach Files to Reservations in Navigator

Some EMS Desktop Client Web Deploy users were unable to attach files to reservations in the Attachments tab of the Navigator.

Fix: Modified functionality so that users can attach files to reservations in the Navigator.
