API

Admin User Requirement Removed When Managing API

As of Update 30, the EMS Room Sign API was moved inside of the Room Sign App. Therefore, customers are no longer required to create an admin user to manage API.

Internal Reference # 41958
(For EMS Internal Use Only)

Check In

Checking into an Exchange-enabled meeting or video conference meeting no longer changes the meeting start time to the time the user checked in

Now, when users check into an Exchange-enabled meeting or video conference meeting from the EMS Room Sign App, the meeting start time no longer changes to the check in time.

This change was made to prevent the following issues: 

  • If the meeting was open in Outlook during check in from the EMS Room Sign App and the meeting start time was changed to the Room Sign app check in time, the meeting ended up with an Outlook rule violation since the meeting's start time was edited while the meeting was in progress.
  • For meetings with multi-location bookings, changing the meeting start time to the check in time from the EMS Room Sign App would update the start times of all the booking locations, not just the location where check in occurred.

Internal Reference # 41365
(For EMS Internal Use Only)

Fixes

AREA DESCRIPTION

LED 

LED Sync Color Issue

When a user created a reservation in front of an existing meeting that was already allowing check-in, the EMS Room Sign App displayed the check-in LED color and the check-in button while displaying the reserved meeting.

Fix: Modified functionality so that the EMS Room Sign App switches to the reserved LED color until the meeting ends and then switches back to the check-in LED status color while hiding the check-in button for the future event.

Internal Reference # 35825
(For EMS Internal Use Only)

Crestron Screen Display

Screen Interface Was Being Cut Off on Crestron Panels

For customers using Crestron panels, portions of the screen interface were being cut off on the following models and resolutions:

  • 7-inch Creston Panel at a resolution of 1024 x 600 (TSS-7-W-S-SIW KIT)
  • 10-inch Crestron Panel at a resolution of 1280 x 800 (TSS-10-B-S-SIW KIT)

Fix: Modified functionality so that the interface displays correctly.

Internal Reference # 41953
(For EMS Internal Use Only)

Time Bar

Clicking Different Hour Sections Selected the Same Start/End Time

When selecting a time in the EMS Room Sign App, clicking a different section in the time bar resulted in the same Start/End time.

Fix: Modified functionality so that clicking on a different hour section results in the selection of a different Start/End time.

Internal Reference # 35993
(For EMS Internal Use Only)

Timebar

Timebar Appeared in Background When Creating a Reservation in Portrait Mode on the Samsung Tab A

When creating a reservation in portrait mode on the Samsung Tab A, users could still see the timebar in the background.

Fix: Modified functionality so that users do not see the timebar when creating a reservation on the Samsung Tab A in portrait mode.

Internal Reference # 31132
(For EMS Internal Use Only)
