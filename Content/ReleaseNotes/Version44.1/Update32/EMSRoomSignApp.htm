AREA DESCRIPTION

LED

LED Sync Error

Users experienced an LED sync error on EMS Room Sign devices when check-in was present at the same time that 'reserve' was also available. When the user created a reservation in front of an existing meeting that was already allowing check-in, the device displayed the check-in color while there was a reserved meeting in progress. Additionally, the 'check-in' button was still displayed for that future event.

Fix: Modified functionality so that the device should switch to the reserved LED color until the meeting ends and then switch back to the check-in LED status color while hiding the check-in button for the future event.

Internal Reference # 35825
(For EMS Internal Use Only)
   
   

Crestron Screen Display

Screen Interface Was Being Cut Off on Crestron Panels

For customers using Crestron panels, portions of the screen interface were being cut off on the following models and resolutions:

  • 7-inch Creston Panel at a resolution of 1024 x 600 (TSS-7-W-S-SIW KIT)
  • 10-inch Crestron Panel at a resolution of 1280 x 800 (TSS-10-B-S-SIW KIT)

Fix: Modified functionality so that the interface displays correctly.

Internal Reference # 41953
