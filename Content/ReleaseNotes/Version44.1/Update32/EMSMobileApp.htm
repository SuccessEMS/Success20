Area

Description

LDAP Groups

 

EMS Mobile App-Enabled Process Templates Assigned by LDAP Groups are not assigned to Everyday User when Authenticating via LDAP using the Mobile App

When utilizing LDAP Authentication on the EMS Web App and assigning process templates based on LDAP Groups, EMS Mobile App-enabled templates were not assigned to the User when utilizing the EMS Mobile App.

Fix: Modified functionality so that EMS Mobile App-enabled templates assigned via LDAP groups should be available on the EMS Mobile App.
