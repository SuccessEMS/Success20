Fixes

Area

Description

Audit Tab

Editing Critical Room Information Was Recorded in the Audit Tab As a Change to the Everyday User Process Template

For rooms tied to an Everyday User Process Template, when a user modified critical room information, such as the room code, the user was incorrectly recorded on the Audit tab as changing the Everyday User Process Template.

Fix: Modified functionality so that the Audit tab for Everyday User Process Templates only records changes made to Everyday User Process Templates.

Internal Reference # 40153
(For EMS Internal Use Only)

Notifications

Notifications Not Triggering

Notifications were not being triggered to targeted recipient when a booking time changed.

Fix: Modified functionality so that notifications are triggered when a booking time changes.

Internal Reference # 40311
(For EMS Internal Use Only)

Email

Unable to Send Emails Using Gmail SMTP Email Settings

Users were unable to send emails from EMS Desktop Client when using Gmail SMTP email settings. When trying to send an email, they would get the error message:

"The SMTP server requires a secure connection or the client was not authenticated. The server response was: 5.5.1 Authentication Required."

Fix: Modified functionality so that users can send emails from EMS Desktop Client when using Gmail SMTP email settings.

Internal Reference # 41280
(For EMS Internal Use Only)
