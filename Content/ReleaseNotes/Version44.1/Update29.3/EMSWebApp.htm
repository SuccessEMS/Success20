Skype for Business

Skype for Business URL and Text Added

Skype for Business URL and meeting information has been added to information sent via the EMS Web App.

Internal Reference # 41485
(For EMS Internal Use Only)

Enhanced Formatting

The font, size, and color of Skype for Business information that appears in the EMS Web App and in the native Skype for Business add-in now match exactly.

EMS Internal Reference # 41489
(For EMS Internal Use Only)
 

Region-specific Dial-in Information Provided

Skype for Business information generated through the EMS Web App now includes ALL local numbers. Skype for Business information generated through the native Skype add-in provides only a few local numbers and a hyperlink to a full phone number list.

EMS Internal Reference # 41482
(For EMS Internal Use Only)
 

Option for Added Disclaimer Text

Option is now available to add HTML-supported disclaimer text to the Skype for Business information generated through the EMS Web App.

EMS Internal Reference # 41543
(For EMS Internal Use Only)
