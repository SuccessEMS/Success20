Floor plans missing in floor plan utility if room name or room code has a backslash. 

Pre-Requisites: ##info##

1. Create a room with name containing backslash sign

2. Create Floor plan Diagram in building with that new room from step 1.

3. Open Floor Plan from step 2 in Floor Plan Configuration Utility.

4. Place there new Floor Plan Indicator for the room from step 1 and save it.

5. Try to re-open Floor Plan Diagram.

6. Also seeing if the room name or code has a backslash the floorplan does not even show in the configuration utility (U21) by removing the backslash, the config utility shows the floor plan

Actual Results: Floor Plan Diagram is not displayed and errors (Uncaught SyntaxError: Unexpected token R in JSON at position xxxxx) appears in F12 developer's console.

Expected Results: Floor Plan Diagram must be displayed with previously saved indicators.
