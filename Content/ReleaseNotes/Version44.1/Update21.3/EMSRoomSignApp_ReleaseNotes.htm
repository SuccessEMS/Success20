Fixes

Area

Description

Parameters

Max Number of Minutes for New Booking Parameter Could Be Overriden

When users set a number for the Max Number of Minutes to Extend Booking parameter, that parameter could be overriden by clicking the Adjust button.

Fix: Modified functionality so that the Max Number of Minutes for New Booking parameter could not be overriden.

Internal Reference # 30959
(For EMS Internal Use Only)

Profile Settings

 

Cancel and Apply Buttons Were Not Displaying in Portrait or Landscape View

The Cancel and Apply buttons were not displaying in the EMS Room Sign App when changing profile settings.

Fix: Modified functionality so that profile settings window should expand to display the cancel and apply buttons.

Internal Reference # 33209
(For EMS Internal Use Only)
