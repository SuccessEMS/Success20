Enhancements

Area

Description

Timezones

 

 

Avoid Database Calls to Get Building Data for Timezone

Instead of the booking search using LookupTimeZoneForBuilding to get timezone information for a booking, EMS database will now utilize the building timezone ID.

EMS Internal Reference # 43097
(For EMS Internal Use Only)

Parameters

Avoid Re-collecting Template Parameter Data

Booking search will retain template parameters for each booking so that previously retrieved data will not be re-collected.

EMS Internal Reference # 43098
(For EMS Internal Use Only)

Booking Rules

 

Concurrent Booking Rule Validation

Booking rule validation will now take place concurrently.

EMS Internal Reference # 43100
(For EMS Internal Use Only)

Booking Search

Concurrent Entity Creation in Booking Search

Entity creation in booking search will now take place concurrently.

EMS Internal Reference # 43099
(For EMS Internal Use Only)
