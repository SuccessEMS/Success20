EMS for Microsoft Outlook Add-In Everyday User Guide

The EMS for Microsoft Outlook add-in provides one-click access to self-service room reservation and resource booking from within the familiar Outlook personal scheduling interface. Users can find available rooms, reserve them, and book resources—such as A/V equipment, catering, etc.—all from within Microsoft Outlook.

This guide includes information on the following topics:

  • Get Started
  • Microsoft Outlook, EMS for Outlook, and VEMS Comparison
  • Create Reservations
    • Single Reservation
    • Series Reservation
    • Video Conference Reservation
    • Edit or Cancel a Reservation
  • Create a List of Favorite Rooms
  • Using Skype for Business in EMS for Outlook
  • Resolve a Booking Conflict
  • View Known Errors/Alerts
