Installation Instructions

To install Master Calendar, you will need to do the following:

  1. Verify you have met the system prerequisites and requirements
  2. Obtain and download the installation files.
  3. Run the Master Calendar Installer
  4. Configure Master Calendar to Use the Master Calendar API Web Service
  5. Install/upgrade the Database Component in Master Calendar
  6. Install/upgrade the Master Calendar Web Service
  7. (Optional) Verifying the Master Calendar API
