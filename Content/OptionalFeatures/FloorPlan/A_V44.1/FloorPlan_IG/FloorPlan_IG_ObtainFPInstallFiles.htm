Obtain Floor Plans Installation Files

The latest Floor Plan Module components can be downloaded from the EMS Customer Portal. Log in and download the following files (required for both first-time installations and upgrades).

  • Floor Plan Web Service (EMSRoomDiagrammingService.msi)
    Installation file for the Floor Plan Web Service – a mechanism used by Adobe Flash Player within the Floor Plan Configuration Utility/ EMS Web App/EMS Kiosk and the web server to render floor map images.
  • Floor Plan Configuration Utility (EMSFloorPlanConfiguration.msi)
    Installation file for the EMS Floor Plan Configuration Utility – a web application used to map indicators to room locations on your floor plans.
