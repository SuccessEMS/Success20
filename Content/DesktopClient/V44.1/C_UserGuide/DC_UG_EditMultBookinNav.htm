Edit Multiple Bookings in the Navigator

This topic guides you in using the wizards available from the Navigator to make the same changes to multiple bookings at once. To edit billable information for a booking, see Editing Charges for Multiple Items.

This topic will also provide information on:

  • Changing the Booking Date/Time
  • Changing Booking Status
  • Changing Rooms
  • Changing Miscellaneous Information

Edit Multiple Bookings

  1. Open the Navigator.
  2. Select the booking(s) on the left pane that you wish to apply the same change(s). The Tools menu offers several wizards to help you batch-edit the bookings with the same changes.
